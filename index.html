About Me

Hello my name is Kevin Nguyen and I am a third year computer science major at the University of California, Riverside. My interests include education and interactive learning research. I am currently part of a project that involves creating a test bank of questions for lower division computer science classes.

Projects

  • Created a customized bash command shell: Rshell
    • Rhsell is an interactive program that mimics the funcitonality of a bash shell with linux commands. Some commands include ls, mkdir, echo, clear, and running bash scripts.
    • Rshell also has a customized functionality of chaining commands using || or && where commands would execute depending on the connector used and whether the inital command executed or not
  • Created a car turn signal simulation using a Basys2 Board:
    • Using Xilinx and a Basys2 FPGA Trainer Board, I created a car turn signal sumulation
    • Using three buttons as input, the furthest leftmost input would blink lights in sequential order to the left, The same would be applied to the rightmost button. The middle button simulated a hazard light that would blink the eight lights on and off.
  • Custom Guitar Hero:
    • Functionality includes a joysitck used for strumming, a LED matrix for falling lights to simulate notes, a LCD display that outputs menu and score, and 16-input keypad for input, four buttons for user interface, and a piezo buzzer for music that was hand transcribed from sheet music to pitch frequency.
